Scoring Types

All or Nothing All correct choices must be chosen to achieve the full score. If a mistake is made the participant achieves no point at all.
Proportional with Penalty Each correct answer provides the participant with a fraction of the total score. If a mistake is made, the fraction is subtracted from the achieved points. For example the total score is 2 and there are 4 options. Each correct answers brings the participant 0,5 points. Each wrong answer subtracts 0,5 points.
